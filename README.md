# Subtitle-Assistant
# 🎬 字幕助手 (FastSubtitle)

一款基于 Whisper AI 的全离线视频字幕生成工具。专为隐私和效率设计，无需上传视频，本地完成识别。

## ✨ 功能特点
- **完全离线**：内置 OpenAI Whisper Small 模型，不消耗流量，保护视频隐私。
- **高识别率**：中文识别准确度极高，能够自动处理标点与断句。
- **极致体验**：支持视频拖拽、白天/黑夜主题切换、SRT 字幕自动导出。
- **绿色免安装**：无需配置 Python 环境，下载解压即可运行。

## 🚀 快速开始
1. 在右侧 **[Releases](https://github.com/Nokunan/你的仓库名/releases)** 页面下载最新的压缩包。
2. 解压 `极速字幕助手.zip`。
3. 双击运行 `极速字幕助手.exe`。

## 🛠️ 开发者环境
如果你想自行编译或运行源码：
- Python 3.10+
- 依赖项：`pip install PySide6-Essentials faster-whisper`
- 需要将 `ffmpeg.exe` 放置在项目根目录的 `ffmpeg/` 文件夹下。
